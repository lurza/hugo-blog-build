Chrome Dev Summit 2020 Dag 2

10 Dec 2020

Dit zijn de must-watch talks van dag twee. Het evenement op dag 2 duurde iets minder dan 4 uur.

Extending CSS with Houdini

Deze talk heeft me veel hoop gegeven voor de toekomst van CSS. Browsers proberen hun interne werking bloot te leggen zodat developers zelf functionaliteit kunnen toevoegen. Hopelijk betekent dit betere cross browser compatibility voor nieuwe CSS-features of creatieve nieuwe properties. Een masonry layout zonder Javascript is voor mij ook al goed :)

Vanaf nu ga ik af en toe naar https://ishoudinireadyyet.com/ surfen.

Transitioning to modern Javascript

Browsers staan niet stil en blijven nieuwe Javascript features toevoegen. Helaas moeten we rekening houden met oudere browsers. Dit wil zeggen dat onze leuke nieuwe features transpiled en polyfilled worden naar tragere, langere, oudere syntax.

De oplossing die ze hiervoor aanbieden is twee bundles maken. Een voor oude en een voor nieuwere browsers.

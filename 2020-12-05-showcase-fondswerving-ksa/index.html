Showcase: Fondswerving KSA

05 Dec 2020

KSA Izegem moet verhuizen! Om wat reclame te maken voor onze fondswerving heeft een oud-leider een affiche gemaakt. Het was aan mij om deze om te zetten naar een website die we gemakkelijk kunnen delen op onze sociale media kanalen. Het is niet gemakkelijk om een affiche te vertalen naar HTML en CSS, maar toch ben ik zeer tevreden van het resultaat! De website kun je vinden op nieuwgebouw.ksaizegem.be.

Om het te delen op sociale media moet er natuurlijk in de preview een aantrekkelijke tekst en foto aanwezig zijn. Sociale media gebruikt meta tags om dit te bepalen. Meer hierover kun je vinden op https://css-tricks.com/essential-meta-tags-social-media/

Must-Try: Alternatieve front end frameworks

22 Feb 2021

Als we aan een nieuw project beginnen grijpen we al rap naar onze vertrouwelijke front end frameworks. Jquery, React, Bootstrap of Vue zijn de perfecte match voor grote, complexe projecten.

Maar wat met kleinere projecten die al deze onnodige functionaliteit niet nodig hebben? Er zijn enkele frameworks die ons de voornaamste features aanbieden met maar een fractie van de bundle size.

FrameworkBundle Size (minified + gzipped)
React42,2 kB
jQuery30,4 kB
Vue.js22,9 kB
Alpine.js8.3 kB
Svelte1,6 kB

Svelte

Svelte is een Javascript framework die geen virtual DOM gebruikt. In plaats daarvan compileert het de UI-componenten naar heel efficiënte, imperatieve code die de DOM rechtstreeks manipuleert. Door de compile stap worden features die je niet gebruikt ook niet meegestuurd naar de browser.

https://svelte.dev/

Alpine.js

De documentatie van Alpine.js zegt: Think of it like Tailwind for JavaScript. Het heeft dezelfde reactive en declarative eigenschappen zoals Vue of React, maar met een veel kleinere bundle size. De syntax lijkt zeer sterk op die van Vue.

https://github.com/alpinejs/alpine

Chrome Dev Summit 2020 Dag 1

09 Dec 2020

Omdat het dit jaar online moest doorgaan, was het formaat er ook aan aangepast. De talks waren divers en telkens maar een korte 10 minuten. Het evenement op dag 1 duurde ongeveer 4 uur. Het was dus bijna niet mogelijk om je te vervelen. Hier zijn mijn favoriete topics van de eerste dag.

Lighthouse

Lighthouse is performance audits rechtstreeks ingebouwd in Chrome. Je laat het enkele seconden runnen en het geeft je een lijst terug van mogelijke performance optimalisaties. Ik ga deze tool zeker gebruiken bij mijn eigen websites.

Core Web Vitals

Er zijn heel veel methodes om performance te meten op een website. Soms net iets teveel methodes waardoor het een angstaanjagende taak wordt om performance te debuggen.

Het Chrome team heeft de belangrijkste er voor ons uitgekozen. De drie Core Web Vitals:

  1. First Contentful Paint: Meet de laadsnelheid
  2. First Input Delay: Meet interactiviteit
  3. Cumulative Layout Shift: Meet de zichtbare stabiliteit

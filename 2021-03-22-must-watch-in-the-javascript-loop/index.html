Must-Watch: In The (Javascript) Loop

22 Mar 2021

Als er een talk is die ik aan iedere Javascript developer zou tonen is het deze. De talk gaat over de browser’s event loop. De event loop is een oneindige lus waarbij de Javascript engine wacht op taken om uit te voeren. Door dit te kijken begrijp ik veel beter wanneer bepaalde Javascript code uitgevoerd zal worden. Alles wordt uitgelegd met behulp van animaties. Deze animaties maken de concepten veel begrijpelijker om te begrijpen dan op papier.

Jake Archibald is een geniale spreker. Als je meer van hem wilt zien dan is zijn show HTTP203 samen met Surma een echte aanrader.

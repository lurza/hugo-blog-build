Must-Watch: Inspiratie voor mijn Bachelorproef

05 Apr 2021

Mijn bachelorproef gaat over de invloed van compatibility op een website. De nadruk ligt op het web toegankelijk maken voor iedereen.

Alhoewel ik veel met Javascript bezig ben wist ik maar weinig over Web Workers. Waarschijnlijk omdat de tools die je van de browser krijgt het je niet gemakkelijk maken. In de presentatie “The main thread is overworked & underpaid” toont surma de Comlink library. Ondertussen heb ik deze zelf nodig gehad bij Shared Workers. Deze talk heeft me dus een hoop extra werk bespaard!

Mijn tweede inspiratiebron is de talk met de prachtige titel World Wide Web, Not Wealthy Westerners' Web. Bruce Lawson haalt ons uit onze westerse bubbel en toont ons mensen met dezelfde noden, maar zonder ons razendsnel internet en moderne gsm’s.

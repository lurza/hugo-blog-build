Showcase: Blog

03 Dec 2020

Het is niet gemakkelijk om aan een blog te beginnen. Er zijn gigantisch veel technologieën waar je er een mee kunt maken. De meeste zijn compleet overkill voor wat ik nodig heb. Een CMS-systeem zoals Wordpress zorgt enkel maar voor extra load time. Het enige wat deze blog moet doen is tekst in een aantrekkelijk formaat op je scherm brengen. Nergens is er Javascript voor nodig.

Dit is dan ook een ideale use-case voor een static site generator. Met de SSG Hugo hoef ik enkel maar markdownbestanden met content te maken. Er is helemaal geen sprake van een database of PHP. De generator maakt van de markdownbestanden statische webpagina’s die op veel plaatsen goedkoop of zelfs gratis gehost kunnen worden. De gegenereerde bestanden zijn niets meer dan wat HTML en CSS. De grootste bestanden op deze site zijn de 100KB aan fonts die ik gebruik. Vertaling: hij is razendsnel.

Het kleurenthema is gebaseerd op Gruvbox Material. Als het programmeren uitlekt tot een lange nacht dan switch ik altijd naar dit thema. Zo bescherm ik mijn ogen en tegelijk ook mijn nachtrust.

Alle code kun je vinden op mijn github: https://github.com/lurza/hugo-blog

Idee: Excel SQL

01 Mar 2021

Dit is een van mijn vele leuke ideeën waar ik nog geen tijd voor heb gehad. Stuur me een berichtje als je het idee samen met mij wilt uitwerken.

Soms krijg ik een CSV of Excel bestand waarop ik een simpele SQL query wil op uitvoeren. Vooralleer ik dan aan query kan beginnen moet ik een SQL server runnen en mijn data importeren.

Door de opkomst van Web Assembly is het ondertussen mogelijk om een volledige SQLite server te runnen in de browser. Het idee is om hierbij een import/export tool bij toe te voegen. Gebruikers kunnen dan gemakkelijk hun data importeren naar de website, er een SQL query op uitvoeren en de data dan weer exporteren.
